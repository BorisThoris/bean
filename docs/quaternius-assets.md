# Quaternius Asset Import

The runtime room uses a curated subset of Quaternius' Modular Sci-Fi Megakit.

- Source: https://quaternius.com/packs/modularscifimegakit.html
- Download page: https://quaternius.itch.io/modular-sci-fi-megakit
- License: Creative Commons Zero v1.0 Universal, https://creativecommons.org/publicdomain/zero/1.0/
- Imported location: `Assets/models/quaternius/modular_sci_fi/`

Only selected FBX files are committed. The full ZIP and extracted source pack are intentionally not kept in the repo.

The game still creates and sizes the room at runtime. These models are composed into repeated floor tiles, modular wall bays, and essential gameplay station parts. Decorative runtime clutter is intentionally excluded so the world stays readable and asset-driven. Physical interaction stays on runtime colliders.
